# Customer Onboarding Checklist

**Project**: NexusPOS AI
**Phase**: Pilot Launch

This checklist is for the NexusPOS Deployment Team when bringing a new restaurant onto the platform.

## 1. Account Provisioning
- [ ] Create Tenant record in the database.
- [ ] Create Outlet record linked to the Tenant.
- [ ] Provision Owner Admin account in Supabase Auth and link to Tenant.

## 2. Discovery & Setup
- [ ] Collect Store Logo and upload to Supabase Storage.
- [ ] Collect Legal Entity Name and GSTIN/Tax ID.
- [ ] Configure Global Tax Rates in the system.
- [ ] Set Timezone and Currency settings.

## 3. Menu Digitization
- [ ] Create Menu Categories (e.g., Starters, Mains, Drinks).
- [ ] Upload Menu Items with pricing.
- [ ] Create Raw Ingredients in the Inventory module.
- [ ] Link Recipes to Menu Items for automatic stock deduction.

## 4. Hardware Deployment
- [ ] Install POS Terminal(s) at front counter.
- [ ] Install Kitchen Tablet(s) in protective casing.
- [ ] Configure Thermal Printer and ensure receipts print correctly.
- [ ] Ensure devices are on a secure, private Wi-Fi network.

## 5. Training Delivery
- [ ] Deliver Owner Manual & walk through Analytics Dashboard.
- [ ] Deliver Cashier Manual & run 2 test transactions (including a split payment).
- [ ] Deliver Kitchen Manual & demonstrate how to bump a ticket.
- [ ] Provide physical copies of the Quick Start Guide.

## 6. Go-Live Handover
- [ ] Confirm Owner has the IT Support Hotline number.
- [ ] Confirm Owner understands the Bug Report Template.
- [ ] Schedule the "Week 1 Check-in" call.
