# Hardware Requirement Guide

**Project**: NexusPOS AI
**Phase**: Pilot Launch

For optimal performance during the NexusPOS AI pilot, participating restaurants must meet the following hardware minimums.

## 1. Point of Sale (Front Counter)
The primary device used by Cashiers to process transactions.
*   **Device Type**: Windows Desktop/All-in-One, Mac, or iPad Pro.
*   **Operating System**: Windows 10/11, macOS 12+, or iPadOS 16+.
*   **Browser**: Google Chrome or Microsoft Edge (latest versions only). Safari is supported but Chrome is recommended for best thermal printer compatibility.
*   **RAM**: Minimum 8GB (16GB recommended for heavy lunch rushes).
*   **Connectivity**: Ethernet (hardwired) connection is strongly recommended over Wi-Fi for stability.

## 2. Kitchen Display System (KDS)
The device used by kitchen staff to view and bump tickets.
*   **Device Type**: Tablet (iPad or Android). Minimum screen size 10 inches.
*   **Protection**: Must be housed in a splash-proof, grease-resistant industrial casing.
*   **Mounting**: VESA wall mount or secure counter stand.
*   **Audio**: Must have functioning internal speakers (for new order chimes) set to maximum volume.
*   **Power**: Must remain plugged into a dedicated power source at all times.

## 3. Receipt Printer
*   **Format**: 80mm Thermal Receipt Printer.
*   **Connectivity**: USB is preferred for single-terminal setups. Ethernet/LAN is required if multiple POS terminals print to a single printer.
*   **Compatibility**: ESC/POS compatible printers (e.g., Epson TM-T88VI, Star Micronics TSP143).

## 4. Barcode Scanner (Optional)
Required only for retail/packaged goods scanning.
*   **Type**: 2D Omnidirectional or standard 1D laser scanner.
*   **Connectivity**: USB Plug-and-Play.

## 5. Network Infrastructure
*   **Bandwidth**: Minimum 25 Mbps download / 10 Mbps upload.
*   **Router**: Dedicated business-class router (e.g., Ubiquiti, Cisco).
*   **Isolation**: The POS and KDS devices should run on a separate SSID (network name) hidden from public/customer Wi-Fi.
