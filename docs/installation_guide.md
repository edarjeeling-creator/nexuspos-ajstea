# Installation Guide

**Project**: NexusPOS AI
**Phase**: Pilot Launch

NexusPOS is a cloud-native Progressive Web App (PWA). There is no heavy local server installation required. Follow these steps to "install" the POS on your front-counter terminal.

## Step 1: Operating System Preparation
1. Ensure Windows/macOS is fully updated.
2. Disable any aggressive sleep/hibernate modes. The POS terminal should be set to "Never Sleep" while plugged in.
3. Install the latest version of Google Chrome.

## Step 2: Printer Setup
1. Unbox the thermal receipt printer and connect it to the terminal via USB.
2. Install the manufacturer's printer driver.
3. Open Windows **Settings > Devices > Printers & Scanners**.
4. Select the thermal printer and click **Manage > Set as default**.
5. Print a test page to verify connectivity.

## Step 3: PWA Installation (Desktop)
1. Open Google Chrome.
2. Navigate to your specific tenant URL (e.g., `https://ajstea.nexuspos.ai`).
3. Log in with your Admin credentials.
4. Look at the right side of the Chrome URL address bar. Click the **Install NexusPOS** icon (it looks like a monitor with a downward arrow).
5. Click **Install**.
6. NexusPOS will now open in its own dedicated window without browser tabs, behaving exactly like a native Windows/Mac application.
7. Right-click the new NexusPOS icon in your taskbar/dock and select **Pin to Taskbar** for easy daily access.

## Step 4: KDS Installation (Tablet)
1. Turn on the Kitchen Tablet and connect to the secure business Wi-Fi.
2. Open Chrome (Android) or Safari (iOS).
3. Navigate to `https://[your-tenant].nexuspos.ai/kds`.
4. Log in with the Kitchen Staff credentials.
5. **Android**: Tap the 3 dots in Chrome -> "Add to Home Screen".
6. **iOS**: Tap the Share button in Safari -> "Add to Home Screen".
7. Open the app from the newly created Home Screen icon to run the KDS in full-screen kiosk mode.
