# Cashier User Guide

Welcome to NexusPOS! This guide covers your daily operations at the front counter.

## 1. Taking an Order
1. From the **Dashboard**, click on the **POS** terminal.
2. Select an **Order Type**: Dine-In, Takeaway, or Delivery.
3. If Dine-In, input the **Table Number**. If Takeaway, input a **Token Number**.
4. Browse categories on the left or use the search bar to find menu items.
5. Tap an item to add it to the cart on the right. Tap again to increase the quantity.

## 2. Managing the Cart
*   **Remove Items**: Click the Trash icon next to any cart item to remove it.
*   **Modify Quantity**: Use the `+` and `-` buttons in the cart to adjust quantities.
*   **Order Notes**: Add a note to the order (e.g., "Allergy: Peanuts", "Extra spicy") using the Notes field at the bottom. This note will print on the receipt and appear on the Kitchen Display System (KDS).

## 3. Processing Checkout
1. Review the Grand Total.
2. If splitting payments, enter the amount tendered for the first method (e.g., CASH), click Add Payment, then process the remaining balance with the second method (e.g., CARD).
3. Once the **Balance Due** hits $0.00, click **Complete Order**.
4. A receipt preview will pop up. You can print this directly to the thermal printer connected to your device.

## 4. Kitchen Communication
Once you click "Complete Order", the ticket is instantly sent to the **Kitchen Display System (KDS)**. You do not need to physically walk tickets to the kitchen unless requested by management.
